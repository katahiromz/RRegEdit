## How to build?

Please use ReactOS Build Environment (RosBE).

```bash
git clone https://github.com/katahiromz/RRegEdit
cd RRegEdit
cmake -G Ninja -DCMAKE_BUILD_TYPE=Release .
ninja
strip regedit.exe
```

Tested on RosBE and Visual Studio 2019.
